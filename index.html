Git

Trabalhando em equipe com controle e segurança

Um curso que expora os benefícios de utilixar o Git como ferramenta de controle de linguagem, em qualquer plataforma.

Principais Benefícios:

  • Funciona de maneira distribuida
  • Permite a edição concorrente de arquivos do projetos
  • Não depende de uma conexão ativa com um servidor
